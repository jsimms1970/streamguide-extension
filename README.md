# StreamGuide - Where to Watch

A Chrome extension that instantly shows you where to stream any movie or TV show.

![StreamGuide Widget](https://img.shields.io/badge/streaming-50%2B%20services-green)
![Chrome Extension](https://img.shields.io/badge/platform-Chrome-blue)
![License](https://img.shields.io/badge/license-MIT-lightgrey)

## Features

- **Automatic Detection** - Widget appears when you visit movie/TV pages
- **50+ Streaming Services** - Netflix, Hulu, Prime Video, Disney+, HBO Max, and more
- **Multiple Options** - Subscription, rent, buy, and free options
- **Non-Intrusive** - Close anytime, remembers your preference
- **Privacy Focused** - No data collection, no tracking

## Supported Sites

- IMDB (`imdb.com/title/...`)
- Rotten Tomatoes (`rottentomatoes.com/m/...` and `/tv/...`)
- Google Search (movie/TV searches)

### Manual Installation (Developer Mode)
1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" (top right)
4. Click "Load unpacked"
5. Select the extension folder

## Usage

1. Visit any movie or TV show page on a supported site
2. The StreamGuide widget appears in the bottom-right corner
3. See all streaming options at a glance
4. Click any service to watch

Click the extension icon in your toolbar to search manually.
